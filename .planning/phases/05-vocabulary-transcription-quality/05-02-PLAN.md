**1. PHANTOM_PATTERNS blocklist** — known Whisper hallucination strings (case-insensitive matching):

```python
PHANTOM_PATTERNS = [
    # English phantoms (most common Whisper hallucinations)
    "thank you for watching",
    "thanks for watching",
    "please subscribe",
    "please like and subscribe",
    "don't forget to subscribe",
    "see you in the next video",
    "see you next time",
    "if you enjoyed this video",
    # Dutch/German phantoms (relevant for NL/Limburgish audio)
    "ondertiteling door",
    "ondertitels door",
    "met dank aan",
    "vertaald door",
    "bewerkt door",
    "untertitel von",
    # Silence/filler phantoms
    "you",
    "...",
]
```

Note: The single-word "you" phantom should only match when it's the entire transcription output (not as substring). Implement this by checking if the stripped text equals "you" exactly.

**2. detect_repetitions(text, min_repeat=5)** function:

```python
def detect_repetitions(text: str, min_repeat: int = 5) -> list[dict]:
    """Detect consecutive word repetitions.

    Strategy: Split text into words. Scan for runs of identical words
    (case-insensitive) >= min_repeat. Also detect repeated phrases
    (2-3 word n-grams repeated >= min_repeat times consecutively).

    Returns list of hallucination dicts with type="repetition".
    """
```

Implementation approach:

- Split text into words (whitespace)
- Scan for consecutive runs of the same word (case-insensitive): if count >= min_repeat, flag as repetition
- For n-gram repetition: check 2-word and 3-word consecutive repeats
- Return empty list if no repetitions found
- CRITICAL: "ja ja dat klopt ja" has max 2 consecutive "ja" — below threshold of 5, so NOT flagged

**3. detect_phantoms(text)** function:

```python
def detect_phantoms(text: str) -> list[dict]:
    """Detect known Whisper phantom strings.

    Strategy: Case-insensitive substring match against PHANTOM_PATTERNS.
    Special case: single-word phantoms (like "you") only match when they
    are the entire text (stripped).

    Returns list of hallucination dicts with type="phantom".
    """
```

**4. detect_hallucinations(text)** — combines both detectors:

```python
def detect_hallucinations(text: str) -> list[dict]:
    """Run all hallucination detectors on text.

    Returns combined list of all detected hallucinations.
    Each dict has: type, matched, start, end
    """
    if not text or not text.strip():
        return []

    results = []
    results.extend(detect_repetitions(text))
    results.extend(detect_phantoms(text))
    return results
```

**5. clean_hallucinations(text)** — removes detected content:

```python
def clean_hallucinations(text: str) -> str:
    """Remove hallucination content from text.

    - Phantom strings: remove the matched substring
    - Repetitions: reduce to max 2 occurrences of the repeated word/phrase
    - Collapse resulting multiple spaces/newlines
    - Strip leading/trailing whitespace
    """
```

**6. process_transcription(text)** — full pipeline:

```python
def process_transcription(text: str) -> dict:
    """Detect and clean hallucinations in one call.

    Returns:
        {
            "cleaned_text": str,
            "hallucinations": list[dict],
            "was_modified": bool
        }
    """
    hallucinations = detect_hallucinations(text)
    if not hallucinations:
        return {
            "cleaned_text": text,
            "hallucinations": [],
            "was_modified": False
        }
    cleaned = clean_hallucinations(text)
    return {
        "cleaned_text": cleaned,
        "hallucinations": hallucinations,
        "was_modified": cleaned != text
    }
```

The module should have NO external dependencies beyond Python stdlib (re, string). Keep it lightweight for <100ms processing.

File structure: module docstring, PHANTOM_PATTERNS constant, detect_repetitions, detect_phantoms, detect_hallucinations, clean_hallucinations, process_transcription. All functions with type hints.
