    **IMPORTANT UX decision (per D-06):** Keep streaming tokens to user as-is during stream. After stream completes per chunk, validate the accumulated output. If JSON is valid, the corrected text is in `validated.corrected`. If fallback, the raw text is used. The current SSE streaming UX remains unchanged from user's perspective — they see tokens flowing.     **HOWEVER:** Since the LLM now outputs JSON (e.g., `{"original":"...","corrected":"..."}`), the user would see raw JSON tokens streaming. To avoid this, for the final pass (not cleanup pass), we need to:
    - Accumulate all tokens for each chunk silently (don't emit individual tokens)
    - Parse the JSON result
    - Extract `corrected` field
    - Emit the corrected text as a single `{type: 'token', text: corrected_text}` SSE event

    This means the streaming behavior changes slightly for Limburgish: instead of token-by-token streaming, each chunk's corrected text arrives as a block. This is acceptable because:
    - Chunks are max 400 words (~2-5 seconds processing time)
    - User still sees progressive output (chunk by chunk)
    - JSON is hidden from user (better UX than raw JSON tokens)

    **3. Add helper functions for glossary formatting and few-shot formatting:**

    ```typescript
    function formatGlossary(region: string): string {
        const glossary = DIALECT_GLOSSARIES[region] || DIALECT_GLOSSARIES['limburgs'] || {};
        if (Object.keys(glossary).length === 0) return '';
        const lines = Object.entries(glossary).map(([k, v]) => `${k}=${v}`);
        return 'Dialect vertaalsleutel:\n' + lines.join('\n');
    }

    function formatFewShotExamples(region: string): string {
        const examples = FEW_SHOT_EXAMPLES[region] || FEW_SHOT_EXAMPLES['limburgs'] || [];
        if (examples.length === 0) return '';
        let formatted = 'VOORBEELDEN (volg dit formaat exact):\n\n';
        examples.forEach((ex, i) => {
            formatted += `Voorbeeld ${i + 1}:\n`;
            formatted += `Input: ${ex.input}\n`;
            formatted += `Output:\n${JSON.stringify(ex.output, null, 2)}\n\n`;
        });
        return formatted;
    }

    const JSON_INSTRUCTION = `OUTPUT FORMAT:
    Geef je antwoord terug als een JSON object met deze structuur:
    {
      "original": "<originele tekst>",
      "corrected": "<gecorrigeerde tekst in standaard Nederlands>"
    }

    Geef ALLEEN het JSON object terug, geen andere tekst.`;
    ```
